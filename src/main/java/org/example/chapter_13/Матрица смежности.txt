Матрица смежности представляет собой двумерный массив, элементы которого
обозначают наличие связи между двумя вершинами. Если граф содержит N вершин,
то матрица смежности представляет собой массив N × N

Заголовками как строк, так и столбцов являются метки вершин. Ребро между
двумя вершинами обозначается 1; при отсутствии ребра элемент матрицы равен 0.
(Также можно использовать логические значения true/false.)

В этой
матрице «соединение» вершины с самой собой обозначается 0, поэтому диагональ,
идущая из левого верхнего в правый нижний угол (от A–A до D–D), заполнена
нулями. Элементы диагонали не несут полезной информации, поэтому в них с таким же успехом можно записать 1,
если это окажется удобнее в вашей программе.

Соответственно при добавлении ребра в граф в матрице смежности необходимо
изменить два элемента вместо одного.
