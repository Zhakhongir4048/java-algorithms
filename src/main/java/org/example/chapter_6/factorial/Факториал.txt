Факториалы сходны с треугольными числами, если не считать того, что вместо
сложения используется операция умножения.
Треугольное число с номером n вычисляется суммированием треугольного числа n – 1 с n,
а факториал n вычисляется умножением факториала n – 1 на n.
Иначе говоря, пятое треугольное число вычисляется по формуле 5 + 4 + 3 + 2 + 1,
а факториал 5 вычисляется по формуле 5 × 4 × 3 × 2 × 1 = 120.

Факториал 0 по определению равен 1.
Для вычисления факториала можно использовать рекурсивный метод, аналогичный методу triangle().
Он выглядит примерно так:

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}
