1) Структурой данных называется способ организации данных в памяти компьютера или в дисковом файле.
2) Правильный выбор структуры данных заметно повышает эффективность программы.
3) Примеры структур данных — массивы, стеки и связанные списки.
4) Алгоритм представляет собой процедуру выполнения некоторой операции.
5) В Java алгоритм обычно реализуется методом класса.
6) Многие структуры данных и алгоритмы, описанные в этой книге, часто используются для построения баз данных.
7) Некоторые структуры данных используются как инструменты программиста:они упрощают реализацию алгоритма.
8) Другие структуры данных моделируют ситуации из реального мира, например
телефонные линии, проложенные между городами.
9) Базой данных называется совокупность данных, состоящих из множества однотипных записей.
10) Запись часто представляет объект из реального мира, например работника или
деталь машины.
11) Записи делятся на поля. В каждом поле хранится одна характеристика объекта,
описываемого записью.
12) Ключом называется поле записи, используемое для выполнения некоторой операции с данными,
например сортировки записей работников по полю LastName.
13) В базах данных можно найти все записи, у которых ключевое поле содержит
заданное значение. Такое значение называется ключом поиска.
