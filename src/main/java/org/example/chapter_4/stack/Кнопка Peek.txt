Вставка и извлечение элементов — две основные операции со стеком. Тем не менее
в некоторых ситуациях бывает полезно прочитать значение с вершины стека без его удаления.
Нажав кнопку Peek несколько раз, вы увидите, как значение элемента Top
копируется в текстовое поле Number, но сам элемент остается в стеке.
Обратите внимание: «подсмотреть» можно только значение верхнего элемента.
Архитектура стека такова, что все остальные элементы остаются невидимыми для
пользователя
