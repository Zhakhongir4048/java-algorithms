Алгоритм поиска посещает один узел на каждом уровне. Следовательно, чтобы оценить длительность
выполнения операции, необходимо знать количество уровней в дереве.
Время выполнения стандартных операций с деревом пропорционально логарифму N по основанию 2.
В O-синтаксисе время выполнения таких операций обозначается O(log2N).
