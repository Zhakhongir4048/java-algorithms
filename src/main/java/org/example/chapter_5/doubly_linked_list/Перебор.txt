Два метода, имена которых начинаются с display, демонстрируют перебор в двусвязном списке.
Метод displayForward() не отличается от метода displayList() обычного связанного списка.
Метод displayBackward() похож на него, но он начинает перебор с последнего элемента и перемещается
к началу списка по значению поля previous каждого элемента. Следующий фрагмент кода показывает, как работает
этот процесс:
Link current = last; // От начала списка
while(current != null) // Перемещение до конца списка
 current = current.previous; // Переход к предыдущему элементу
