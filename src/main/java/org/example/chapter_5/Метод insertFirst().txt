Чтобы вставить в список новый элемент, достаточно присвоить полю next созданного объекта ссылку
на предыдущий первый элемент, а затем изменить поле first так,
чтобы оно указывало на только что вставленный элемент.

Выполнение метода insertFirst() начинается с создания нового элемента на
основании данных, переданных в аргументах. Затем ссылки изменяются так, как
говорилось ранее
