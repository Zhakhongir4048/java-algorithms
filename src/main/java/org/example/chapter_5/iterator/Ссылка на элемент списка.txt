Соответственно будет проще предоставить пользователю возможность создать столько ссылок, сколько ему
необходимо. Наиболее естественным решением для объектно-ориентированного языка будет внедрение
каждой ссылки в объект класса. Это не может быть класс списка, так как объект списка существует
в одном экземпляре, поэтому обычно ссылка реализуется в отдельном классе.
