Структуры данных определяют способ организации данных в памяти компьютера (или на диске).
Алгоритмы обеспечивают выполнение различных операций с этими структурами.

Алгоритмы — это основа программирования, определяющая, каким образом разрабатываемое
программное обеспечение будет использовать структуры данных
