public void selectionSort() {
    int out, in, min;
        for(out = 0; out < nElems - 1; out++) { // Внешний цикл
            min = out; // Минимум
                for(in = out + 1; in < nElems; in++) { // Внутренний цикл
                    if(a[in] < a[min]) // Если значение min больше,
                        min = in; // значит, найден новый минимум
                }
            swap(out, min); // Поменять их местами
        }
}

Внешний цикл (переменная out) начинает перебор с начала массива (индекс 0) и продвигается к большим
значениям индексов.
Внутренний цикл (переменная in) начинается с out и также двигается вправо.
В каждой новой позиции in сравниваются элементы a[in] и a[min].
Если a[in] меньше, то min присваивается значение in. В конце внутреннего цикла min указывает на минимальное
значение текущего прохода, и алгоритм переставляет элементы массива, на которые ссылаются out и min.
