Программа в отличие от нас не «видит» данные. Она может сравнивать только
двух игроков, потому что операторы сравнения работают именно по такому принципу

Все три алгоритма, представленные в этой главе, состоят из двух шагов, которые
повторяются снова и снова, пока данные не будут отсортированы:
1. Сравнить два элемента.
2. Поменять элементы местами или скопировать один из них.
Тем не менее разные алгоритмы по-разному выполняют эти операции
